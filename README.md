# Soia Kotlin Client

Library imported from Kotlin and Java code generated from soia files.

## Installation

### Gradle (Kotlin DSL)

```kotlin
dependencies {
    implementation("land.soia:soia-kotlin-client:1.1.4")  // Pick the latest version
}
```

### Gradle (Groovy DSL)

```groovy
dependencies {
    implementation 'land.soia:soia-kotlin-client:1.1.4'  // Pick the latest version
}
```

### Maven

```xml
<dependency>
    <groupId>land.soia</groupId>
    <artifactId>soia-kotlin-client</artifactId>
    <version>1.1.4</version>
</dependency>
```

## Java Compatibility

This library can be used in Java projects as well as Kotlin projects. The generated code and client library are fully compatible with Java.
